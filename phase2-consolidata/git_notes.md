# 📝 Git 极简笔记

## 🛠️ 常用命令
* `git status`：查看当前工作区状态
* `git add .`：暂存所有修改过文件
* `git commit -m "提交信息"`：提交到本地仓库（信息须明确）
* `git push origin main`：推送代码到远程仓库

## 📌 规范原则
1. **防脏提交**：必须配置 `.gitignore`，严禁提交 `__pycache__` 等缓存垃圾。
2. **隐私保护**：明文密码、手机号、API Key 等敏感文件坚决不上云。
3. **小步快跑**：每完成一个独立功能（如：清洗完一个文件）就 commit 一次。
